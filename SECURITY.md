# Security policy

## Supported versions

Only the latest release is supported with security fixes.

## Reporting a vulnerability

Please do not open a public issue for a suspected vulnerability. Use GitHub's
private vulnerability-reporting flow for this repository, or contact the
maintainer through the repository profile with a concise report and steps to
reproduce. You should receive an acknowledgement within seven days.

Please avoid including credentials, access tokens, or other sensitive data in
the report.
