Useful when quiet. Ready when needed.

Codex Minibar reads usage from a locally installed, authenticated Codex CLI or Desktop app and keeps limits visible in the Windows notification area. Native WinUI 3. No browser tab. No credential handoff.

Everything in the tray

Every setting applies immediately — no relaunch required to change widgets, notifications, or startup behavior.

Configurable tray widgets

Show the five-hour limit, weekly limit, both, or the primary reset clock. Stack multiple widgets in the tray and pick the presentation that reads at a glance.

  • Stacked numbers, bars, or nested rings for both limits
  • Single number, bar, or ring for one window
  • Reset time or countdown for the five-hour window
  • Remaining or used percentage, per widget

Compact usage popup

Open a native popup for the current plan, credits, limit windows, and local usage history — without digging through another app.

Smart notifications

Optional Windows toasts for limit resets, low session or weekly remaining usage, approaching reset, activation results, Codex unavailable, and new app updates.

Automatic activation

Optionally start Codex to activate a fresh five-hour window when a small nudge is all you need.

Usage pace

See whether consumption is ahead of or behind a steady burn across the current window, right where you already look.

Start with Windows

Register for login startup so limits are already visible when the session begins — toggle it off anytime in Settings.

In-place updates

Check GitHub Releases on startup and install a matching portable package without rebuilding your setup. Disable checks whenever you want.

Local history

Keep usage history on disk with retention from 1 to 365 days. Settings and history stay in your Windows user profile.

Codex discovery

Detects Codex CLI or Desktop automatically. Override with a custom executable path when your install lives somewhere unusual.

Popup preferences

Show used percentage, surface usage pace, or hide plan and credits from the popup when you want a quieter view.

Install options

Per-user NSIS installer (no admin) or portable zip. Pick x64 or ARM64 to match your machine.

Your credentials are none of its business.

Codex Minibar talks to the local Codex app server from an already authenticated installation. It does not ask for, store, or transmit your Codex credentials.

Settings and usage history remain in your Windows user profile. Update checks hit GitHub Releases only when you leave that option enabled.

Keep your limits in sight.

Free and open source. Download the installer or portable build for Windows, then find Codex Minibar in the notification area.
